# How to Create Daily Report

2018/1/25

Notice

As part of our daily workflow this year 2018, all are required to make daily report according to what tasks/research you did during the day every 4:55 PM. But you can prepare the note every morning when you time in.

SETUP

  1. We recommend to use Evernote as your notes will be saved in the cloud for free. Sign up with your Binary email address to official.
  2. Next, create a Notebook that say's "Daily Reports".
  3. Share this your "Daily Reports" notebook to your immediate supervisor's email and the following emails: rmstdiego@binaryideas.com and rluna@binaryideas.com.

DAILY REPORT

You will be given by your immediate supervisor the tasks you will have for the day. Of course there will also be pending tasks that you're still working from the past days. We will include past, present and future tasks on your daily report. To summarize the daily report, we need a format to construct it.

Note: If you type "[]" (open and close brackets) and hit space in evernote, it will create a checkbox. This box is good to indicate that a specific tasks is done.

Let me explain how a specific task is read from a report. The tasks should have font size of 14. Guides first,

[] ==> checkbox - check this box only if the task is done.
[John Doe] ==> (bold and italicized) assigned person (do this if you're a supervisor to determine which tasks are for whom in your team.
[40%] ==> (bold and italicized) percentage of the task
{these are the comments} ==> (italicized) comments for each task. Optional but helpful for report reviewers to have a quick status/update on task.

Format is:

PROJECT NAME (bold and capitalize)
[checkbox] [task title] [assigned person] [percentage] [comments] - [link from the PMS]

Example:

CRBC
[ ] NextGen Website Updates [Riel Stephen] [70%] {Updated User Interface} - http://pms.binaryideas.com/#/tasks/13020522

In Evernote, you can see it like this:

That's for each tasks.

DETAILED REPORT

Now that you know the format of each task report, we can now construct the whole report for the day.

  1. Create a note inside your "Daily Reports" notebook for each day (early in the morning). The title should be "Daily Report - [Day] [Day]". Ex. "Daily Report - January 25, 2018".
  2. Next have 3 sections on your note: "YESTERDAY", "TODAY", "OVERTIME" and "TOMORROW" with 36 font size.
  3. In "YESTERDAY", list all tasks here you did yesterday.
  4. In "TODAY", list all the tasks you will do today and update the percentage, etc.
  5. In "OVERTIME", list all tasks that scheduled for overtime (approved by your immediate supervisor).

  6. In "TOMORROW", list all the tasks that you will continue for the next day or you have scheduled tasks for tomorrow. You can copy and paste the tasks from "TODAY" if you have to continue those tomorrow. Your report should somewhat look like this

  7. Once your report is ready. Share this note to your immediate supervisor's email and to rmstdiego@binaryideas.com and rluna@binaryideas.com

  8. The next morning, you can just duplicate your report from yesterday and move tasks so you can save time making the report again.

PS: This will be a living guide. Meaning, this document can be updated anytime.
